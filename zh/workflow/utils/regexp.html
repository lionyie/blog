格式相关 ​

344 格式手机号 ​

  • 从左到右
js
/(^\d{3}|\d{4}\B)/g
// 例：
'15512341234'.replace(/(^\d{3}|\d{4}\B)/g, '$1 ')
1
2
3
  • 从右到左
js
/\B(?=(\d{4})+\b)/g
// 例：
'15512341234'.replace(/\B(?=(\d{4})+\b)/g, ' ')
1
2
3

隐藏手机号中间 4 位 ​

js
/(\d{3})\d{4}(\d{4})/
// 例：
'15512341234'.replace(/(\d{3})\d{4}(\d{4})/, '$1****$2')
