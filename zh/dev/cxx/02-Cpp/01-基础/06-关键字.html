sizeof关键字 ​

① sizeof关键字可以统计数据类型所占内存大小。

② sizeof关键字，语法：sizeof ( 数据类型 / 变量 )

python
#include <iostream>
using namespace std;

int main()
{

    cout << "short 类型所占内存空间为：" << sizeof(short) << endl;

    system("pause");

    return 0;

}
